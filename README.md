PNU Intelligent Chatbot (Rasa + React) 🎓🤖

این پروژه یک دستیار هوشمند آموزشی است که برای پاسخگویی به سوالات متداول دانشجویان مهندسی کامپیوتر دانشگاه پیام نور طراحی شده است. این چت‌بات با استفاده از قدرت پردازش زبان طبیعی (NLP) به دانشجویان در رفع ابهامات آموزشی کمک می‌کند.

🚀 ویژگی‌های کلیدی (Key Features)

هسته هوشمند: استفاده از Rasa Open Source برای درک زبان طبیعی (NLU).

داده‌های بومی: آموزش مدل با بیش از ۲۰۰۰ نمونه جمله فارسی (رسمی و محاوره‌ای).

دقت بالا: کسب دقت ۹۹.۹٪ در تشخیص Intentها.

رابط کاربری مدرن: طراحی شده با React و Tailwind CSS.

ارتباط Real-time: اتصال پایدار از طریق REST API به سرور Rasa.

🛠 تکنولوژی‌های مورد استفاده (Tech Stack)

Rasa: چارچوب یادگیری ماشین برای گفتگوهای متنی.

React.js: کتابخانه جاوااسکریپت برای ساخت رابط کاربری.

Tailwind CSS: برای طراحی زیبا و ریسپانسیو.

Python: منطق سمت سرور و پردازش‌های پشتیبان.

📦 نحوه اجرا (Installation & Setup)

۱. اجرای بخش Rasa

ابتدا کتابخانه‌های مورد نیاز را نصب کرده، مدل را آموزش داده و سپس سرور را اجرا کنید:

# نصب Rasa
pip install rasa

# آموزش مدل
rasa train

# اجرای سرور با دسترسی CORS
rasa run --cors "*" --port 5056


۲. اجرای بخش فرانت‌بند (React)

در پوشه مربوط به کدهای فرانت‌بند، دستورات زیر را وارد کنید:

# نصب بسته‌های npm
npm install

# اجرای پروژه
npm start


📝 ساختار پیکره زبانی (Corpus Structure)

پیکره زبانی این پروژه در قالب فایل‌های YAML طراحی شده و شامل دسته‌بندی‌های زیر است:

[x] امور آموزشی و درسی: سوالات مربوط به سرفصل‌ها و قوانین آموزشی.

[x] سیستم گلستان: راهنمای انتخاب واحد و کار با سامانه.

[x] امور مالی: سوالات مربوط به شهریه و پرداخت‌ها.

[x] تعاملات عمومی: احوال‌پرسی و پاسخ‌های عمومی.
